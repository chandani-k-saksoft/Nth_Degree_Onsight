3. bddDriver Class Documentation

The bddDriver class is a comprehensive test driver for handling various types of tests including web, API, mobile, and desktop. It uses different drivers and tools to manage these tests and provides functionalities for test setup, execution, and teardown.

Class Overview

The bddDriver class includes methods for initializing and managing different types of drivers, handling test reports, and logging test results. It also supports accessibility compliance checks and performance testing.

  • @After("@API") - Cleans up after API tests and updates the test results.
